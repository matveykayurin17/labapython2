Данная лабораторная работа реализует мини-оболочку с командами: ls,cat,cd,cp,mv

Как это работает?:

    1-Пользователь вводит команду
    2-В функции parse 1ая функция разбивает строку, введённую пользователем на два списка,один из которых содержит в себе команду/путь, а другой флаги. Затем вторая функция обрабатывает строку, введёную пользователем и вызывают функцию самой команды(если команды не существует или пользователь что-то неправильно ввёл, то функция вернут ошибку
    3-Команда реализуется в файле functions
Какие функции вообще есть?:

    1-ls(выводит содержимое каталогов)
    2-cd(переходит в указанную директорию)
    3-mv(переименовывает/перемещает файл, перемещает каталог)
    4-cat(возвращает содержимое файла)
    5-cp(копирует файл из одного каталога в другой/рекурсивно копирует каталог)
